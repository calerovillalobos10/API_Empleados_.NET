# Proyecto API Web (ASP.NET Core Web API)

Este proyecto es una API web construida con ASP.NET Core y Entity Framework Core. Proporciona una estructura básica para desarrollar servicios web robustos y escalables.

## Tecnologías Utilizadas

- **ASP.NET Core**: Framework para construir aplicaciones web modernas y de alto rendimiento.
- **Entity Framework Core**: ORM (Object-Relational Mapper) para trabajar con bases de datos relacionales.
- **SQL Server**: Base de datos relacional utilizada.

## Paquetes NuGet

- `Microsoft.EntityFrameworkCore`
- `Microsoft.EntityFrameworkCore.SqlServer`
- `Microsoft.EntityFrameworkCore.Tools`

## Configuración del Proyecto

### Instalación de Dependencias

Asegúrate de tener instalado .NET Core SDK. Para instalar los paquetes necesarios, puedes utilizar el siguiente comando:

```sh
dotnet add package Microsoft.EntityFrameworkCore
dotnet add package Microsoft.EntityFrameworkCore.SqlServer
dotnet add package Microsoft.EntityFrameworkCore.Tools
